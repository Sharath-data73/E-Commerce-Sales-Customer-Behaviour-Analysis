# E-commerce Data Analysis Using Python

Welcome to the **E-commerce Data Analysis** project — an end-to-end data analytics project focused on transforming raw e-commerce data into actionable business insights through **Data Cleaning, Exploratory Data Analysis (EDA), Business Intelligence, and Visualization** using Python.

---

## Project Overview

This project analyzes an e-commerce dataset to uncover valuable insights that support business growth and strategic decision-making. The workflow covers:

* Data Cleaning & Preparation
* Exploratory Data Analysis (EDA)
* Customer & Sales Insights
* Profitability Analysis
* Visualization & Reporting

The goal is to improve data quality, understand sales patterns, identify business opportunities, and optimize profitability.

> *"Raw data becomes powerful when transformed into meaningful business intelligence."*

---

## Data Cleaning Steps

### Step 1: Understanding the Dataset

* Checked missing values
* Verified data types
* Reviewed dataset dimensions
* Analyzed column structure

### Step 2: Handling Missing Values

* Customer Segment → Filled using mode
* Profit & Shipping Cost → Filled using median

### Step 3: Removing Duplicates

* Eliminated repeated transactions

### Step 4: Data Type Conversion

* Converted sales, profit, shipping cost to numeric
* Converted order and shipping dates to datetime

### Step 5: Logical Validation

Removed:

* Negative sales values
* Invalid shipping dates

### Step 6: Outlier Treatment

* Shipping Cost → 99th percentile capping
* Profit → IQR method

### Step 7: Data Standardization

Standardized:

* Product Category
* Region
* State names
